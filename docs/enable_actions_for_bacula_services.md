<h2 align="center">Bacula - Enable actions for `bacula` services</h2>

- Login into https://bacularis.myfirma.de/web/login/

- Menu => API Panel => Settings => Actions => Enabled => Hack in

![Enable actions](https://github.com/johann8/bacularis-ubuntu/raw/master/docs/assets/screenshots/enable_services_actions.png)

-  Fill out all the commands below => Save

![Add actions commands](https://github.com/johann8/bacularis-ubuntu/raw/master/docs/assets/screenshots/add_actions_command.png)


- Check bacula service `director`:  Menu => Director


![Add actions commands](https://github.com/johann8/bacularis-ubuntu/raw/master/docs/assets/screenshots/check_service_director.png)


- Check bacula service `service`: Menu => Storage => File1 => Details


![Check services commands](https://github.com/johann8/bacularis-ubuntu/raw/master/docs/assets/screenshots/check_service_storage.png)
